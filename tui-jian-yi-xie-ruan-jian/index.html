推荐一些软件

2020.03.29   共 699 字

背景

在 2020 年 3 月 29 日，我难以忍受电脑的卡顿，决心优化电脑，卸载一些无用庞大的软件，更新一波 APP，然后重塑一个新的环境。

我的行动开始于斩除一个名为 alibabaprotect 的服务，在此过程中我发现一些好的软和方法，特在此记录。

推荐的 win10 软件

火绒

下载地址

利用火绒的强力粉碎文件，我清除了 alibabaprotect 所在的目录。

Ccleaner

下载地址

直接运行 Ccleaner 清理垃圾，我就多出了 5GB 的 C 盘空间。

Everything

下载地址

基于名称快速定位文件和文件夹。

Bandicam

原始链接地址

百度网盘地址，密码 se39

百度网盘地址，密码 vs3t

我需要一个录屏软件，以后我或许会用此录制视频上传 B 站，今日特下载，在此推荐。

FreeMove

Github

下载地址

免安装的快速软件迁移工具，我把巨大的 dotnet 从 C 盘移至 E 盘，C 盘遂空出 3GB。

天若 OCR

微盘下载地址

一款我很喜欢的免费 OCR 软件，扫描很快，体积也很小。

ResEditor

微盘下载地址

一款可以修改 EXE 文件详细信息的软件，很好用。

TreeSize Free

下载地址

可以查看一个目录下文件的大小，对磁盘优化很有帮助。

Registry Workshop

官方下载地址

破解下载地址

我的电脑注册表编辑器被禁用了，我懒得下载组策略(win10 家庭版)，直接用这个就行了。
