简介

项目仓库

项目文档

Math Complex 是一个采用 TypeScript 编写的 JavaScript 扩展数学库，它扩展了 Math 数学类的一些方法，让 JavaScript 支持复数计算。

基础功能

  • 扩展了复数类
  • 实现了复数类的四则、乘方开方运算
  • 扩充了辐角、指数转换计算
  • 通过 Reflect 静态类实现函数式
  • 函数式对原生 number 类型兼容

泽火革卦解析

2019.04.17   共 2577 字

原题： 除旧布新 从易经“泽火革”卦看发展变革之道

引言

“泽火革”是易经中的第四十九个卦象，其上卦为泽兑，下卦为火离，根据自然现象，我们也能注意到，火与泽会相互影响，使得当前的现象不断发生变化，难以稳定在一个稳定的状态。

由此我们就可以引申出其“变革”、“发展”、“变化”的含义。

变化其实本就是易经的根本智慧，“革”本身就是一个不断发展的卦象，每个卦之间都是在相互影响，彼此相应，而正是其自我的激励才让整个自然环境生生不息勃然发展。

对于“革”卦，如果我们可以认真学习其中的无上智慧，博达至理，我们就能在困境中寻找出变革之法。 古语云：“穷则变，变则通，通则久”，在灰暗的环境中披荆斩棘是困难的，但是变革也是必须的，痛苦也是需要忍耐的。

对于当下社会环境，在大力推荐改革的氛围下，理解活用革卦的理念就显得更加重要，这也是本文的社会意义所在。
